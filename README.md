Аренда бытовых инструментов «ToolRental»

«Аренда бытовых инструментов» — это Android-приложение, которое объединяет каталог доступных инструментов с системой бронирования, управления объявлениями и пользовательскими отзывами. Проект направлен на упрощение поиска и аренды инструментов для домашних и профессиональных задач, создание сообщества пользователей и предоставление актуальной информации о доступных инструментах.

Основные цели:
- Структурированный каталог инструментов с фильтрацией по категориям и цене;
- Удобный поиск инструментов по названию и описанию;
- Система бронирования с выбором дат и расчетом стоимости;
- Возможность сдавать инструменты в аренду;
- Объективные рейтинги владельцев на основе отзывов арендаторов;
- Управление объявлениями (создание, редактирование, удаление);
- Просмотр запросов на бронирование и их подтверждение;
- Администрирование контента;
- Ролевая модель: Гость, Арендатор, Владелец.

Технологии
- Язык программирования: Java
- Среда разработки: Android Studio
- База данных: Firebase (Realtime Database / Firestore, Authentication).
